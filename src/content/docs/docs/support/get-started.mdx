What does Web Monetization give content supporters?
* The ability to control whether to pay, how much to pay, and how often to pay
* The ability to pay in their preferred currency
* An easy way to stream and/or tip, once set up, without requiring the user to enter billing details for every transaction
* A platform-agnostic way to support creators
