This page provides basic instructions for setting up and sending Web Monetization payments.

<Steps>

1. Sign up with a [digital wallet](/docs/wallet-address/wallets) that supports Web Monetization payments and get your assigned wallet address. For example:
    ```http
    https://wallet.example.com/alice
    ```
2. <mark>Download and install the Interledger Web Monetization extension</mark>.
3. <mark>Configure the extension.</mark>
4. Visit some of our <mark>recommended sites</mark> to get started with Web Monetization!

</Steps>
